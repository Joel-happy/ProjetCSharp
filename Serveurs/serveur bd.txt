Documentation du Projet : Mise en Place d'un Serveur MySQL avec Docker
A. Installation de Docker sur Debian
Mise à jour des paquets :

Exécuter sudo apt update et sudo apt upgrade pour mettre à jour le système.
Installation de Docker :

Installer Docker avec sudo apt install docker-ce.
Vérifier l'installation avec sudo systemctl status docker.
B. Configuration du Conteneur MySQL
Téléchargement de l'Image MySQL :

Utiliser docker pull mysql pour télécharger l'image officielle MySQL.
Création du Conteneur MySQL :

Lancer le conteneur avec docker run --name mysql-container -e MYSQL_ROOT_PASSWORD=mon_mot_de_passe -p 3306:3306 -d mysql.
Le mot de passe root est défini à mon_mot_de_passe.
Configuration de Redémarrage Automatique :

Utiliser docker update --restart always mysql-container pour configurer le conteneur afin qu'il redémarre automatiquement.
C. Exécution du Script SQL
Copie du Script SQL dans le Conteneur :

Copier le script depuis la machine hôte vers le conteneur avec docker cp /chemin/vers/script.sql mysql-container:/chemin/dans/conteneur.
Exécution du Script :

Se connecter au conteneur avec docker exec -it mysql-container bash.
Accéder à MySQL avec mysql -u root -p.
Exécuter le script avec source /chemin/dans/conteneur/script.sql.
D. Vérification et Tests
Vérification des Tables :

Après l'exécution, vérifier la création des tables avec SHOW TABLES; dans MySQL.
Tests Fonctionnels :

Effectuer des opérations de base comme l'insertion et la sélection de données pour tester la fonctionnalité.
E. Conseils et Bonnes Pratiques
Sauvegardes Régulières : Mettre en place des routines de sauvegarde pour la base de données.
Surveillance : Surveiller la performance et la santé du conteneur et de la base de données.
Sécurité : Assurer la sécurité du conteneur et de la base de données.
